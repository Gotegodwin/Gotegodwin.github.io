# Nkeeh Baridosian Godwin — Premium Portfolio

A modern, dark-themed Web3-inspired personal portfolio built with **HTML + Tailwind CSS (CDN) + Vanilla JavaScript**.

No frameworks, no build step. Just open `index.html` in any browser.

## Features

- Fully responsive (mobile-first)
- Sticky navigation + mobile hamburger menu
- Glassmorphism cards
- Soft blue/purple gradients & glowing accents
- Scroll-triggered fade-in animations
- Floating background blobs
- Smooth scrolling
- SEO meta tags + accessible markup
- Easy to customize (replace content without touching design)

## File Structure

## How to Replace Content (No Design Changes Needed)

### 1. Your Photo
- Place your professional photo as:  
  `assets/images/profile.jpg`  
- Recommended: square or portrait, at least 600×600 px, optimized (< 300 KB).
- The site already points to this path. If the image is missing it shows a nice initials fallback.

### 2. Resume / CV
- Replace the file:  
  `assets/resume.pdf`  
- Keep the exact filename `resume.pdf`.
- The “Download CV” and “View Resume” buttons + embedded viewer will automatically use the new file.

### 3. Portfolio Projects
- Open `index.html` and find the **Portfolio** section (search for `id="portfolio"`).
- Each project is a card with:
  - Image `src` (change to `assets/images/your-project.jpg`)
  - Title
  - Short description
  - Category tag
  - Link (change the `href` of the “Open” button)
- You can duplicate a whole card `<div class="portfolio-card ...">` to add more projects.

### 4. Video Samples
- Find the **Video Samples** section (`id="videos"`).
- Replace the YouTube embed `src` URLs with your real video embed links.
- Update the title and short description under each card.
- Thumbnail is automatically generated by YouTube; no extra image needed.

### 5. Contact Links
- Search for the Contact section or the social links in the footer / contact cards.
- Replace the `href` values with your real:
  - Email (`mailto:you@email.com`)
  - X / Twitter
  - LinkedIn
  - Telegram
  - GitHub

### 6. Text & Experience
- All text is real content based on the provided CV.
- Edit the paragraphs, timeline items, skills badges, education, and certifications directly in `index.html`.
- Keep the existing CSS classes so the design stays intact.

## Running Locally

1. Download or clone the folder.
2. Open `index.html` in Chrome, Firefox, Safari, or Edge.
3. For best results on mobile, use a local server (optional):
   ```bash
   npx serve .
   ```

## Hosting (Free)

- **Netlify Drop**: drag the whole folder to https://app.netlify.com/drop
- **Vercel**: import the folder or connect a GitHub repo
- **GitHub Pages**, Cloudflare Pages, Surge.sh — all work perfectly.

## Customization Tips

- Colors: edit the Tailwind config inside the `<script>` in `index.html` (look for `tailwind.config`).
- Glow intensity / blur: adjust values in `style.css`.
- Animation speed: change the CSS transition durations or the Intersection Observer threshold in `script.js`.
